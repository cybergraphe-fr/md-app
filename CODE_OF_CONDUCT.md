# Code of Conduct

This project follows the Contributor Covenant.

## Our Standards

- Be respectful and professional.
- Focus on constructive technical discussion.
- Assume good intent, but prioritize evidence and clarity.
- Do not harass, discriminate, or intimidate contributors.

## Enforcement

Report unacceptable behavior to the maintainers via the security contact described in SECURITY.md.

## Scope

This code of conduct applies in project spaces, issue trackers, pull requests, code reviews, and community discussions related to this repository.
